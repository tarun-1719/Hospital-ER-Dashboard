# 🏥 Hospital Emergency Room Dashboard (Excel Project)

## 📌 Project Overview

This project is an interactive Excel dashboard designed to analyze hospital emergency room (ER) data. It helps track patient flow, waiting time, and satisfaction levels to improve operational efficiency.

---

## 🎯 Objectives

* Analyze patient volume trends
* Identify peak hours in ER
* Measure average waiting time
* Track patient satisfaction
* Provide actionable insights

---

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Slicers (for interactivity)

---

## 📊 Key Features

* Interactive dashboard with filters
* KPI tracking:

  * Total Patients
  * Average Wait Time
  * Satisfaction Score
* Trend analysis by time and department

---

## 🔍 Key Insights

* Patient volume is highest during evening hours
* Waiting time increases during peak periods
* Higher waiting time leads to lower satisfaction
* Some departments experience heavier patient load

---

## 💡 Business Impact

* Helps optimize staff allocation
* Reduces patient waiting time
* Improves patient satisfaction
* Supports data-driven decision making
